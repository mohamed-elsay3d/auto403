# auto403

**auto403** is an automated 403 Forbidden bypass testing tool designed for **Bug Bounty hunters** and **Web Pentesters**.

It systematically tests multiple HTTP methods, path tricks, and header payloads to identify misconfigurations that allow access to forbidden resources.

---

## Features

- Automated 403 bypass techniques
- Multiple HTTP methods testing
- Header-based bypass payloads
- Path normalization tricks
- Burp Suite proxy support
- Clean & colored output
- ASCII banner (terminal friendly)
